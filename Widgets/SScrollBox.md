### SScrollBox
#### 支持的参数

* ColorAndOpacity
* Orientation
* Padding
* ToolTipText
* Visibility
* Slots

Example 1:

    "SScrollBox": {
        "Slots":
        [
            {
                "Padding": 5,
                "SVerticalBox":
                {
                    "Slots": [
                        {
                            "padding": 2,
                            "SButton": {
                                "Text": "Button A"
                            }
                        },
                        {
                            "padding": 2,
                            "SButton": {
                                "Text": "Button B"
                            }
                        }
                    ]
                }
    		}
    	]
    }
